# MD Path Mentions Autocomplete

Автокомплит путей файлов и папок проекта по символу `@` в markdown-файлах — удобно для вставки ссылок на файлы в промпты, заметки и документацию.

## Возможности

- Начните вводить `@` в `.md`-файле — появится список файлов и папок текущего workspace
- Продолжайте печатать, чтобы отфильтровать список (например `@src/ext` найдёт `src/extension.ts`)
- Индекс файлов строится один раз при активации и автоматически обновляется при создании/удалении файлов
- Директории `node_modules`, `.git`, `dist`, `out`, `build` исключены из индекса по умолчанию

## Настройки

| Настройка                           | По умолчанию                                                                     | Описание                                                |
| ----------------------------------- | -------------------------------------------------------------------------------- | ------------------------------------------------------- |
| `mdPathMentions.excludeGlobs`       | `["**/node_modules/**", "**/.git/**", "**/dist/**", "**/out/**", "**/build/**"]` | Glob-паттерны, исключаемые из индекса                   |
| `mdPathMentions.maxFiles`           | `8000`                                                                           | Лимит файлов в индексе                                  |
| `mdPathMentions.includeDirectories` | `true`                                                                           | Показывать директории в автокомплите                    |
| `mdPathMentions.maxSuggestions`     | `300`                                                                            | Максимум вариантов в списке за один запрос              |
| `mdPathMentions.trace`              | `false`                                                                          | Подробные debug-логи в Output-канале `MD Path Mentions` |

## Команды

- **MD Path Mentions: Rebuild Index** (`mdPathMentions.rebuildIndex`) — принудительно пересобрать индекс файлов

## Известные ограничения

- Работает только в файлах с языком `markdown`
- `@` внутри email-адресов и подобных конструкций не триггерит автокомплит (срабатывает только после пробела, начала строки или открывающих символов вроде `(`, `[`, `"`)
